# ToyBox Store

A premium, sustainably crafted toy store built with Next.js, Tailwind CSS, and Shadcn UI.

## Features

- **Next.js App Router**: Modern routing and layouts.
- **Tailwind CSS**: Sleek, responsive design.
- **Shadcn UI**: Premium components for a high-end feel.
- **Framer Motion**: Smooth animations and transitions.
- **Cart Management**: Context-based cart system with a slide-out drawer.
- **Responsive Design**: Optimized for all devices.

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/`: Next.js App Router pages and layouts.
- `src/components/`: Reusable React components.
- `src/context/`: React Context providers (Cart, etc.).
- `src/data/`: Static product data.
- `public/assets/`: Static images and assets.
